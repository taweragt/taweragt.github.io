Política de Exclusão de Dados

Última atualização: 17 de julho de 2026

Sobre este aplicativo

O Taiom.Nunes é um aplicativo de integração com a API do Instagram/Meta, usado para gerenciamento de conta e conteúdo do Instagram conectado pelo usuário. Esta página explica quais dados são armazenados e como qualquer pessoa pode solicitar a exclusão completa dos seus dados.

Quais dados armazenamos

  • Token de acesso concedido pelo Meta para operar a conta conectada.
  • Informações básicas de perfil (nome de usuário, ID da conta, foto pública).
  • Metadados de conteúdo necessários para as funcionalidades de gerenciamento (ex.: listas de publicações, métricas públicas).

Não armazenamos senhas do Instagram/Facebook — o acesso é feito exclusivamente via login oficial do Meta.

Como solicitar a exclusão dos seus dados

Duas formas de solicitar

  1. Revogar o acesso pelo Meta: em Instagram/Facebook, acesse Configurações → Aplicativos e Sites → localize "Taiom.Nunes" → remova o aplicativo. Isso envia automaticamente um pedido de exclusão de dados ao nosso sistema.
  2. Solicitar diretamente por e-mail: envie uma mensagem para taweragt@gmail.com com o assunto "Exclusão de dados" e o nome de usuário da conta conectada.

Prazo e confirmação

Após recebida a solicitação, todos os dados associados à conta são removidos em até 30 dias. Uma confirmação é enviada para o e-mail de contato informado na solicitação.

Contato para privacidade

taweragt@gmail.com

Esta política se aplica exclusivamente aos dados obtidos por meio da integração com a API do Meta/Instagram para o aplicativo Taiom.Nunes.
