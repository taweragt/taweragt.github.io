Aplicativo Taiom.Nunes

Política de Privacidade

Última atualização: 17 de julho de 2026

1. Introdução

Esta política explica quais dados o aplicativo Taiom.Nunes coleta ao ser conectado a uma conta do Instagram/Facebook via API oficial do Meta, para que fins são usados, e como você pode controlá-los.

2. Dados que coletamos

  • Token de acesso concedido pelo Meta no momento da autorização.
  • Informações básicas de perfil (nome de usuário, ID da conta, foto pública).
  • Metadados de conteúdo necessários para as funcionalidades de gerenciamento (ex.: listas de publicações e métricas públicas da própria conta).

Não coletamos senhas — o acesso é feito exclusivamente pelo fluxo de login oficial do Meta, e nunca temos acesso à sua senha do Instagram/Facebook.

3. Como usamos os dados

Os dados coletados são usados exclusivamente para operar as funcionalidades de gerenciamento de conta oferecidas pelo aplicativo. Não usamos seus dados para fins publicitários próprios nem os revendemos.

4. Compartilhamento com terceiros

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de marketing. Os dados trafegam apenas entre sua conta, o Meta (via API oficial) e a infraestrutura do próprio aplicativo.

5. Armazenamento e segurança

Os dados são armazenados pelo tempo necessário para o funcionamento do aplicativo e protegidos contra acesso não autorizado. Tokens de acesso podem ser revogados a qualquer momento diretamente nas configurações do Instagram/Facebook.

6. Seus direitos

Você pode solicitar a exclusão completa dos seus dados a qualquer momento. Veja o processo completo na Política de Exclusão de Dados.

7. Alterações nesta política

Esta política pode ser atualizada periodicamente. Mudanças relevantes serão comunicadas através do e-mail de contato associado à conta.
